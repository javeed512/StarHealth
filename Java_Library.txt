Multiple Inheritance:
======================

Inheritance:
Class: In Java  a class can extends only One Class.
Interface: In Java an interface can extends multiple interfaces as parent.


interface I1{

public abstract void m1();

}


interface I2 {

public abstract void m2();

}


interface I3  extends I1 , I2{

public abstract void m3();

}



class  Demo  implements I1 , I2 , I3{

	void m1(){
		//logic
		}
	void m2(){
		//logic
		}
	void m3(){
		//logic
		}


}

class  Test  implements I3{


	void m1(){
		//logic
		}
	void m2(){
		//logic
		}

	void m3(){
		//logic
		}


}
